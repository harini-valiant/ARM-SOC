
This repository contains experiments and scripts for SoC design, HDL simulation, and driver compilation.

## Tools Used

- **Arm GNU Toolchain (AArch64)**
  - Compiler: `aarch64-none-elf-gcc`
  - Source: [Arm Developer ToolChain](https://developer.arm.com/downloads/gnu-toolchain)

- **Icarus Verilog**
  - Simulator: `iverilog`, `vvp`
  - Source: [Icarus Verilog](https://bleyer.org/icarus/)

- **GTKWave**
  - Waveform viewer: `gtkwave`
  - Source: [GTKWave Project](http://gtkwave.sourceforge.net/)
